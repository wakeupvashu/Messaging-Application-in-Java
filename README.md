# Messaging-Application-in-Java
This is a console-based messaging application developed in Java. It allows users to register with unique usernames, send messages to other registered users, and view their inbox. The project demonstrates core Java concepts such as object-oriented programming (OOP), collections (HashMap, ArrayList), and user input handling.
Messaging Application in Java
A simple console-based messaging system built with Java to demonstrate core concepts of object-oriented programming (OOP), collections, and user input handling.

This project allows users to:

Register with a unique username

Send messages to other registered users

View their inbox with received messages

It’s lightweight, runs in the console, and serves as a foundation for future enhancements like GUI, persistent storage, and real-time communication.

 Features
User Registration: Ensures unique usernames

Message Sending: Send text messages between registered users

Inbox Viewing: Displays all received messages for each user

In-Memory Storage: Uses Java Collections (HashMap, ArrayList)

Tech Stack
Language: Java (JDK 17+)

Concepts Used: Classes, Objects, Encapsulation, Collections

Environment: Console-based (IDE optional: Eclipse, IntelliJ IDEA, NetBeans)

 Project Structure
User class → Handles inbox and message receiving

MessagingApp class → Manages user registration, sending messages, and inbox display
